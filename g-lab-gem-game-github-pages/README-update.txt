g-Lab. 宝石ゲーム v84 更新手順

【最終の花設定】
・梅（12%小さく調整済み）
・芍薬
・菊
・牡丹

花は「むつかしい」だけで出現します。
やさしい／ふつうでは花は出ません。

【コピーするもの】
1. app.js
2. service-worker.js
3. assets フォルダ

上記3つを、既存の
g-lab-gem-game-github-pages
フォルダへ上書きしてください。

assets フォルダは「結合」または「すべて置き換え」を選んでください。
flower-assets 内に次の4ファイルが入れば正しい状態です。
・ume.png
・shakuyaku.png
・kiku.png
・botan.png

その後、GitHub Desktopで Commit → Push origin を行ってください。
