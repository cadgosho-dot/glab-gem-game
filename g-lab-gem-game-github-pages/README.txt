遊び方
- ZIPを解凍して index.html を開くと遊べます。
- スマホでは左右に位置を合わせ、タップまたは「落とす」で宝石を落とします。
- 同じ宝石が触れると次の宝石に進化します。
- BGM ON/OFF、効果音 ON/OFF、リセットボタンがあります。

アプリ化メモ
- manifest.webmanifest と service-worker.js を入れているため、Web公開すればホーム画面追加型のPWAとして使えます。
- App Store / Google Play 用にする場合は、このWebゲームをCapacitor等で包む流れが現実的です。
